# Readme : Smart Robotic Arm


## About the project
The main objectif of our project is to manipulate a robot arm ( PincherX 100 ) using simple commands . We are using a Rasbperry Pi 4b as our board , and Ubuntu  as our OS.





## Buit with
- Raspberry Pi 4B
- Ubuntu Mate 20.04
- ROS Noetic
- Python 3
- OpenCv
- Yolov3
